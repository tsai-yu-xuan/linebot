# 使⽤說明:
圖文選單 可點選 "DOG" 或 "CAT"，或是傳文字 "狗" 或 "貓"，
透過文字篩選出 "狗" 或 "貓"，即可回傳該動物的10筆領養資料。
回傳圖文筐會有品種、顏色、年齡、是否開放認養、性別、所屬的收容所名稱，
也有按鈕可以點選"來電詢問"、"地圖" 接可連到指定的電話及google地圖。
